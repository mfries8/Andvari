# Andvari Data Directory Structure

This folder contains all data inputs and outputs for the Swarm.

## Directory Tree

```text
data/
├── raw_training_data/      
│   ├── positive/           <-- Place massive calibration drone images containing proxy meteorites here
│   └── negative/           <-- Place massive calibration drone images containing only dirt/background here
│
├── sliced_training_data/   <-- Input dataset for the Augmenter (Populated by Slicer)
│   ├── positive/           <-- Sliced 224x224 chips containing proxies
│   └── negative/           <-- Sliced 224x224 chips of background noise
│
├── raw_test_data/          <-- Drone SD card dump, used as input for the main pipeline search grid
│
└── output/                 <-- Output directory for results (Generated by the Swarm pipeline)
    ├── thumbnails/                 <-- Cropped 224x224 candidate images created by the Cartographer
    ├── verified_candidates.csv     <-- Raw data file generated by the Cartographer
    ├── verified_candidates.kml     <-- Google Earth map payload generated by the Cartographer
    └── final_deployment_targets.csv<-- Target list of verified hits (Generated by the Auditor UI)
```
